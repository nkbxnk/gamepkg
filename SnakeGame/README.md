# 🐍 Snake Game

เกมงู (Snake Game) พัฒนาด้วย **Python** และ **Pygame**

## 🎮 คุณสมบัติ

- ควบคุมงูด้วยปุ่มลูกศร
- เก็บอาหารเพื่อเพิ่มคะแนน
- คะแนนแสดงบนหน้าจอ
- เกมจบเมื่อชนกำแพงหรือชนตัวเอง
- สามารถรีสตาร์ทเกมได้


## 🛠️ โปรแกรมที่ใช้

- Python 3.x
- Pygame

ติดตั้ง Pygame

```bash
pip install pygame
```

---

## ▶️ วิธีรันเกม

รันด้วย Python

```bash
python snake.py
```

หรือดับเบิลคลิก

```
run_game.bat
```

หรือ

```
dist/snake.exe
```
